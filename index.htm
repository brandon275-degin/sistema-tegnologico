Automatizacion

La Automatizacion es la aplicación de la tecnología, programacion, robótica o procesos que permiten realizar tareas de manera mas eficiente y con minima intervencion humana,reduciendo errores, tiempo y costos.

Sistema tecnologico

Es un conjunto de elementos que trabajan juntos para cumplir un proposito espesifico usando la tecnologia.

Programacion

Es el proceso de elaborar instrucciones claras y precisas, escritas en un lenguaje que la computadora pueda entender, para que esta realice tareas especificas de manera autónoma.
