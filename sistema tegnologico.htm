Elementos que componen un sistema tegnológico

Entrada

La entrada se refiere a los recursos que se suministran al sistema para que pueda funcionar Estos recursos pueden ser materias primas, energía, o información.
Dependiendo el tipo de sistema, las entradas pueden provenir de diferentes fuentes.

Procesos

Son las acciones o tranformaciones que ocurren dentro del sistema para convertir las entradas en salidas.
Incluye tambien la regulación y los mecanismos de control que aseguran que el sistema funcione correctamentey que la salida sea la esperada.

Salida

Es el resultado que entrega el sistema después de procesar la entrada.
