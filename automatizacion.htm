Sistema Automatico y Control

Es un conjunto de elementos que funcionan de manera coordinada para ralizar una tarea sin intervencion constante del ser humano. Están diseñado para ejecutar acciones previamente programadas o responder a estimulos del entorno.
El Sistema de Control:es el encargado de regular el funcionamiento del sistema automático.
Se ocupa de comparar lo que se desea obtener (valor de referencia) con lo que realmente se obtiene (salida real) , y si hay diferencia, realiza ajustes para corregir el proceso.

Elementos que componen un sistema automatico y de control

Sensores

Los sensores son elementos responsables de captar informacion del entorno o del propio sistema. Esta informacion puede incluir temperatura, presision, velocidad entre otros parametros.

Controladores

El controlador es el cerebro del sistema automático. Este dispositivo procesa la información recibida de los sensores y emite órdenes a los actuadores para
realizar las tareas programadas.

Interfaz de Usuario

La interfaz de usuario permite a los operadores interatuar con el sistema automático. A traves de estas interfaces,es posible monitorear el estado del sistema,ajustar parámetros y resolver problemas.

Actuadores

Permiten a un sistema realizar acciones fisicas concretas a partir de una señal de control. En otras palabras, transforman energia eléctrica en movimiento o fuerza.


Tipos de Control

Control Manual

Es la forma mas basica, en el cual el operador interviene directamente para poner en marcha o detener el sistema.
Este tipo de control depende totalmente de la supervision humana, ya que el operador debe vigilar permanentemente el funcionamiento de la maquina

Control Automatico

El control automático es un sistema en el que la intervencion humana pasa a segundo plano.
Un dispositvo controlador toma desisiones por si mismo, midendo la salida del sistema con sensores, comparando con valores deseados, y actuando con actuadores para correguir desviaciones.



Concepto de robots

El robot es una maquina o sistema mecanico-eletrónico programable que puede realizar tareas de manera automática o semiautomática,sustituyendo en parte el trabajo humano.
Puede estar diseñado para manipular objetos, desplazarse, procesar informacion o incluso imitar comportamientos humanos.

Tipos de Robots

Robots industriales

Estos robots estan especialmente diseñados para integrarse en la cadena de producción industrial, donde su función principal es ejecutar tareas repetidas y rutinarias de manera constante y precisa. Gracias a su programación y a su capacidad de operar sin interrupcion, contribuyen a aumentar la eficiencia y reducir errores humano.


Robots de Servicio

Es una máquina programable capaz de trabajar de manera automática o semiautomática, diseñada para realizar tareas útiles fuera de la indistria de manufactura.


Robots Móviles

Son robots capaces de desplazarse y tomar decisiones de manera idependiente, casi en tiempo real,utilizando sensores y un sistema de procesamiento interno para adaptarse al entorno y cumplir sus tareas.


Robots Humanoides

Son un tipo especial de robots,con forma semejante a la humana y diseñados para realizar funciones similares a las de las personas.


Elementos y Sensores basicos de los robots

Estructura Mecánica

La estructura mecánica proporcionara soporte y alojamiento para todos los componentes internos del robot.
Está fabricada con materiales como metales, plastico o fibra de carbono, y su diseño varía según la función y tipo de robot.


Sensores de robótica

Estos dispositivos detectan informacion acerca del entorno, como los:

  • Sensores de Luz:Son dispositivos que dectectan la cantidad de luz en su entorno y permite que un robot perciba cambios de iluminación y reaccione antes estos.

  • Sensores de Aproximacion:Detectan la presencia de objetos cercanos sin necesidad de tocarlos. Se usan para que los robots eviten obstáculos o reconozcan su entorno.

  • Sensor de Distancia:Permite midir que tan lejos esta un objeto respecto al sensor. Esto permite que los robots eviten choques,midan espacios o se posicionen correctamente.

Fuente de energia

Los robots necesitan una fuente de energia para funcionar que puede ser eléctrica, hidráulica, neumatica a una combinacion de estas.
En el caso de los robots móviles y autónomos lo mas comun es el uso de baterias recargables, mientras que los robots industriales utilizan una fuente de energía externa.
